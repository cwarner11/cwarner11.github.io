About Me

Hello, my name is Christopher Warner and I am from Saint Louis, Missouri. I graduated with a Bachelor's degree in French from Clark Atlanta University in 2015 and a Master's degree in French with a concentration in Second Language Pedagogy and Applied Linguistics from Georgia State University in 2018

In September 2019, I moved to Paris, France, to become an English teacher. However, during the Covid-19 crises, while I was in quarantine, I decided to pursue my programming interest. I had no real experience with programming, but upon writing my first "Hello, World," I was hooked.

I started out with HTML and CSS. I liked it, but that was not my goal. I wanted to code. I found C programming and got a solid foundation in arrays, algorithms, memory, and data structures. Finally, there was an introduction to Python and SQL, and Python quickly became my preferred language. Subsequently, I have learned C++, and have excelled in Java.

The quarantine was a blessing in disguise. I couldn't break away from my discovery and often spent my days working on projects or thinking about solutions. I am now a junior at the University of Missouri - St. Louis studying computing technology and working towards a certificate in Artifical Intelligence. My carrer goal is to work as a software developer using artifical intelligence and machine learning.

I look forward to honing my skills and taking this new adventure as far as possible

Projects

Here are some projects that I have recently finished.

Filter

C

This program takes a bitmap image, and a filter as command-line argument and applies the specified filter to the picture. I wrote the code that would take each pixel and transform it using the appropriate filter cod, and the four filter codes required to complete this task: grayscale, sepia, reflection, and blur.

Conversion Calculator

Python, Django, JavaScript, HTML, CSS

This web application allows users the chance to sign up/login to take advantage of three conversion calculators: weight (between pounds and kilograms), length (between miles and kilometers), and temperature (between fahrenheit and celsius).

Readability

Python

This program uses the Coleman-Liau index to determine the readability level of a text. It uses a United States grade level and ranges from "Before Grade 1" up to "Grade 16+" (senior undergraduate student).

Hogwarts Houses

Python/SQL

This program imports student data into an SQLite database, and then it takes the Hogwarts house as a command-line argument. It produces a class roster for each of the houses of Hogwarts: Gryffindor, Slytherin, Hufflepuff, and Ravenclaw.

Time Calculator

Python

This program takes in two required parameters and one optional parameter. I wrote function that adds the duration time to the start time and returns the computed result. If the function is given the optional starting day of the week parameter, the output displays the day of the week of the result.

Inventory

Python

This program is an open-ended problem with no definitive solution. It is an API client that consumes an API that shows the total CPU, memory, and disk space of each group and allows us to limit to top-k results, and checks for group overlap. when given two groups, it prints all nodes that are in both.
